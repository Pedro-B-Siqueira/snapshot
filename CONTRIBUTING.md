# Contributing to snapshot

Improvements to `snapshot/SKILL.md` are welcome — open a PR with before/after examples.

## How

1. Fork the repo
2. Edit `snapshot/SKILL.md` — this is the only file that controls skill behavior
3. Open a PR with:
   - **Before:** what snapshot outputs now
   - **After:** what snapshot outputs with your change
   - One sentence explaining why the change is better

CI will automatically sync your changes to all agent-specific copies.

Small focused change > big rewrite.
